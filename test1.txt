Question 44
Skipped
A data analyst wants to compute the average order_amount per region from the orders table, including only completed orders, with each region listed once.

Which query is correct?

SELECT AVG(order_amount) FROM orders GROUP BY status WHERE region = 'completed';

Correct answer
SELECT region, AVG(order_amount) AS avg_amount FROM orders WHERE status = 'completed' GROUP BY region;

SELECT region, order_amount AS avg_amount FROM orders WHERE status = 'completed';

SELECT region, MEAN(order_amount) FROM orders WHERE status = 'completed';

Overall explanation
The correct solution applies the WHERE filter, then groups by region, and averages order_amount per group. MEAN is not an ANSI SQL aggregate, and lacks GROUP BY. The other solutions swaps the role of GROUP BY and WHERE, which is invalid or returns the raw column, not its average.
